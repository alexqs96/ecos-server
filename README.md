- ### Requiere las siguientes variables de entorno (archivo: .env)
```sh
  MONGO_URI=URI de mongodb
  JWT_SECRET=
  CLIENT_URL=http://localhost:3000
  CLOUDINARY_CLOUD=
  CLOUDINARY_API=
  CLOUDINARY_SECRET=
  NODE_ENV=development
```

- ### Comandos para iniciar el proyecto
```sh
cd huertar_server

npm i

npm run dev
```
